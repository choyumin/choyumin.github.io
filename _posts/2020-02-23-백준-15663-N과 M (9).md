## 15663.  N과 M (9)

---

### Code

```java
import java.io.*;
import java.util.Arrays;
import java.util.Iterator;
import java.util.LinkedHashSet;

public class N과M9 {
	static StringBuilder sb;
	static int N, M;
	static int[] arr;
	static boolean[] visited;
	static int[] sel;
	static LinkedHashSet<String> lhset;

	public static void main(String[] args) throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		BufferedWriter bw = new BufferedWriter(new OutputStreamWriter(System.out));

		String[] s = br.readLine().split(" ");
		String[] str = br.readLine().split(" ");
		lhset = new LinkedHashSet<>();

		N = Integer.parseInt(s[0]);
		M = Integer.parseInt(s[1]);

		arr = new int[N];
		visited = new boolean[N];
		sel = new int[M];
		sb = new StringBuilder();

		for (int i = 0; i < N; i++) {
			arr[i] = Integer.parseInt(str[i]);
		}

		Arrays.sort(arr);
		perm(0);

		Iterator<String> itr = lhset.iterator();

		while (itr.hasNext()) {
			bw.write(itr.next());
			bw.write("\n");
		}
		bw.close();

	}

	private static void perm(int idx) {
		if (idx == sel.length) {
			for (int i = 0; i < sel.length; i++) {
				sb.append(sel[i] + " ");
			}

			lhset.add(sb.toString());
			sb.delete(0, sb.length());
			return;
		}

		for (int i = 0; i < arr.length; i++) {
			if (!visited[i]) {
				visited[i] = true;
				sel[idx] = arr[i];
				perm(idx + 1);
				visited[i] = false;
			}
		}
	}

}

```

### 문제점
중복을 제거하기 위해 LinkedHashSet을 사용했다. 처음에 TreeSet을 사용했는데 테케는 다 맞았지만 제출하면 틀렸습니다가 나왔따.
찾아보니 TreeSet 말고 LinkedHashSet을 사용해야 통과되었다.
TreeSet과 LinkedHashSet의 차이를 정리해서 올려야겠다.
